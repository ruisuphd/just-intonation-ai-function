# Experiment Registry

This folder stores concise experiment records for the AI-enhanced JI project.

Each experiment should be written as a separate markdown file using the template in `TEMPLATE.md`.

## Required Fields

- question
- hypothesis
- code version
- data split
- preprocessing
- model or baseline configuration
- metrics
- hardware context
- main results
- statistical method
- conclusion

## Naming Convention

Use a sortable prefix:

- `YYYY-MM-DD-short-name.md`

Example:

- `2026-03-09-score-free-baseline-v1.md`
